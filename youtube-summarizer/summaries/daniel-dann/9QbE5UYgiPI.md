## Key insights
-   Kamatera is a global cloud infrastructure provider with over 20 data centers worldwide, offering enterprise-level hardware.
-   Servers run on modern Intel Xeon Platinum Ice Lake processors with fast NVME SSD storage.
-   Pricing is flexible with hourly or monthly billing options and no long-term contracts, allowing users to pay only for what they use.
-   New users can access a 30-day free trial worth up to $100 in cloud resources to test the platform.
-   Server configuration is highly customizable ("like a constructor"), allowing users to individually select CPU cores (e.g., 1-2 for simple sites), RAM (e.g., 1-2 GB), and storage capacity (e.g., 30-50 GB, expandable).
-   Users should choose a data center location closest to them or their target users to minimize latency and ensure optimal performance.
-   Operating system choices include Alma Linux, Debian, CentOS, Windows, and Ubuntu (Ubuntu 24.04 LTS recommended for its reliability and long-term support).
-   "Type A availability" servers are shared hardware, ideal for learning and non-critical tasks.
-   Server provisioning is very fast, typically taking less than a minute to make a server active.
-   Kamatera offers automated deployment options for various applications via "Create new app" (e.g., WordPress, Minecraft, LLM environments, Python, Superbase) and "Create new service" for development environments (e.g., Laravel, Kubernetes).
-   "Create new desktop" allows deployment of virtual machines with graphical interfaces (Windows, Ubuntu, Debian, Fedora).
-   Support includes direct access to real engineers and qualified professionals through a ticket system, rather than just chatbots.

## Patterns & frameworks
-   **"Build your server like a constructor"**: A step-by-step methodology for customizing server components (zone, OS image, CPU, RAM, storage, networking) individually, providing granular control over resource allocation.
-   **Location Optimization Principle**: A best practice for choosing the data center geographically closest to the user or target audience to minimize network latency and maximize performance.
-   **LTS (Long-Term Support)**: A software release model, particularly for operating systems like Ubuntu, indicating a version that will receive stable, long-term updates, making it ideal for production environments.
-   **Hourly Billing Model**: A flexible payment framework where users are charged only for the actual time their server is running, allowing for cost efficiency, experimentation, and easy scaling down.
-   **Multi-Tiered Deployment Options**: Kamatera provides several pathways for deploying resources:
    -   Manual server configuration for ultimate customization.
    -   "Create new app" for quick, one-click deployment of common applications.
    -   "Create new service" for specific development environment setups.
    -   "Create new desktop" for complete virtual machines with graphical user interfaces.
-   **30-Day Free Trial (up to $100)**: A common onboarding strategy allowing new users to explore and test the full capabilities of the platform risk-free before making a financial commitment.
-   **Real Engineer Support**: A customer service model that emphasizes direct access to qualified technical professionals for support, rather than relying solely on automated chatbots, for effective problem resolution.
