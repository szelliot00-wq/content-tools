# Replit Agent 4 Is Here: Everything You Need to Know

Video ID: `wpKQJnvy8tQ`

## Summary
This video provides a "no hype review" of Replit Agent 4, showcasing its new capabilities for planning, designing, and building applications using multiple AI agents simultaneously. It highlights three key features: designing multiple variations on an infinite canvas, parallel building with multiple AI agents, and generating diverse outputs like slide decks and animations beyond just apps. The speaker demonstrates building a habit tracker app from a simple prompt, managing AI agents through a Trello-style taskboard, and publishing the full-stack application. The video concludes with an analysis of Agent 4's position in the AI coding market, emphasizing its potential for non-technical users and the belief that code underpins all knowledge work.

## Key insights
- Replit Agent 4 enables users to plan, design, and build multi-component applications with multiple AI agents concurrently, all launching simultaneously.
- **Three core new features** of Agent 4 are:
    1.  **Designing multiple variations** on an infinite canvas (e.g., generating four distinct design variants for a calendar tab: Glass Morphic, RPG Quest, Botanical, and a surprise variant).
    2.  **Building using multiple AI agents in parallel** (e.g., assigning one agent to build the calendar view and another to build the habits tab simultaneously).
    3.  **Going beyond generating apps** to create slide decks, animations, and other documents, based on the principle that "code is the foundation of all knowledge work."
- The demo involved creating a mobile habit tracker app with two tabs (calendar and habits view) and streak tracking from a simple text prompt.
- The **canvas view** allows users to select a preferred design variant and then prompt the agent to generate additional views for the chosen style. The speaker suggested improvements like AI agent cursors and direct manual editing for a more delightful experience.
- The **parallel building feature** includes a **Trello-style taskboard** interface with stages like "Draft," "Active," "Ready," and "Done" to help users (acting as "team leads") manage and monitor the progress of their AI agents.
- Replit's full-stack integration allows for publishing the entire app (frontend, database, backend) with a single click, automatically including a security scan step.
- The generated habit tracker app was approximately "90% working," with minor bugs noted (streaks not functional, text cut off in some areas).
- Agent 4 can transform raw text into formatted outputs like a detailed slide deck or an animated video, demonstrating its ability to create diverse content from code.
- The speaker posits that the AI coding market is segmenting into "tools for engineers and developers" (e.g., Claw Code, OpenAI's Codex) and "tools for everyone else," where Replit Agent 4 is strategically positioned.
- **Replit Agent 4 makes two significant bets**:
    1.  Non-technical individuals (like designers and product managers) will collaborate directly with AI agents to achieve their goals.
    2.  Code is the underlying foundation for all knowledge work, meaning an agent capable of writing code can generate a vast array of digital outputs beyond just applications.
- Overall, Agent 4 is considered a "big step forward" on both these bets, despite some areas (like the canvas experience) still needing more polish.
