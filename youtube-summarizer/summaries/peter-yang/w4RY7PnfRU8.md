# I Watched 6 AI Agents Design an App Together And It Blew My Mind | Tom Krcha

Video ID: `w4RY7PnfRU8`

## Summary
This video showcases "Pencil," an innovative AI product that allows a swarm of AI agents to design anything visually. It highlights how Pencil serves as a "visual planning mode" for generating mobile apps, websites, and marketing materials, emphasizing rapid ideation and exploration of design variations. A core innovation is the "Pen" file format, a platform-agnostic, JSON-based descriptor built for agents, which can then be converted into various codebases like React or Swift iOS. The product aims to humanize AI through visible cursors for each agent and integrates deeply with development tools like Cursor, offering a seamless workflow for designers, developers, PMs, and marketers to build and iterate on ideas quickly.

## Key insights
-   **Pencil Product Overview:** Pencil is a cross-platform (Windows, Linux, Mac) design tool with plugins for VS Code, Cursor, and other IDEs, allowing users to invite AI to design. Users can also plug in their own custom agents or CLIs.
-   **Swarm Mode for Parallel Design:** A recently shipped feature enables multiple AI agents (e.g., six Cloud Opus 4.6 agents) to work in parallel on a single design request. For example, two agents can work on each of three mobile app screens, significantly speeding up the design process.
-   **Humanized AI with Cursors:** Each AI agent is represented by a visible cursor on the canvas, providing a "magic" and humanized feel. This visual feedback helps users understand which agent is working on what, aids debugging, and creates a sense of collaboration.
-   **"Pen" (Platform-Agnostic Design Format):** Pencil uses a proprietary, JSON-based file format called "Pen" (e.g., `coffee shop.pen`). This format is an "agentic descriptor" for design, built for AI agents to easily read and write, serving as a "source of truth" that can live in Git. It can be converted into various outputs like Figma, Lovable, Swift iOS, Kotlin, React Native, or HTML/CSS/JS.
-   **Integrated Design-to-Code Workflow:** Pencil allows users to design visually and then generate full, runnable code (e.g., React, Next.js, Tailwind) directly from the Pen file. This can be done within the Pencil app or integrated into an IDE like Cursor, where the Pen file opens in a visual editor.
-   **Real-time Interaction with Agents:** A key differentiator is the ability to edit the design on the canvas *while* AI agents are actively working. This "canvas workflow" keeps users in the flow, unlike traditional byte-coding apps that require waiting for agents to complete their tasks.
-   **Exploratory Design & Divergence:** Pencil supports generating multiple design variations (e.g., 20) side-by-side, reflecting real-world designer workflows of exploring many options before converging on a final design.
-   **Image & Component Integration:** The tool integrates with various image generators (e.g., Nano Banana) and includes advanced design components (e.g., Chaten) with customizable styles (dark mode, accent colors) and prompt nodes for saving pre-made instructions.
-   **Rapid Adoption & Broad Appeal:** Since its launch 8 weeks prior to the video, Pencil garnered 100,000 users. Its user base extends beyond designers and engineers to include product managers, marketers, solopreneurs, and even enterprises, allowing non-coders to create marketing materials, ads, and technical specs.
-   **Founder's Vision:** CEO Tom Krcha (former Adobe, founder of Around) aims to bring back the fun and accessibility of designing and coding in one place, reminiscent of Flash, by simplifying the complexities of modern development through visual AI.

## Use cases
-   **Mobile App Design:** Rapidly designing multiple screens for mobile applications (e.g., travel logs) with AI-generated layouts and imagery.
-   **Website Creation:** Generating full React/Next.js/Tailwind websites from visual designs, even converting selected frames to light/dark mode with an AI agent.
-   **Marketing & Sales Collateral:** Creating marketing materials, ads, PDFs, and technical specifications for salespeople by visually designing them with AI.
-   **Rapid Prototyping & Ideation:** Using the "visual planning mode" to quickly explore and conceptualize app or website ideas before committing to full development.
-   **Design System Management:** Converting existing enterprise design systems into the Pen format to create a "source of truth" that lives in Git and can be used by AI agents and human collaborators.
-   **Design Iteration & Exploration:** Generating numerous design variations (e.g., 20 different styles) for a component or screen to facilitate divergence and convergence in the design process.
-   **Developer Handoff:** Providing clear, agentic design files (Pen format) to developers, easing the transition from design to code.
-   **Learning & Empowerment:** Enabling individuals with no prior coding experience to visually design and generate code, fostering a new generation of "makers."
-   **Debugging AI Agent Workflows:** Observing the cursors of individual agents to understand their parallel actions and debug issues in real-time.
-   **Component Library Creation:** Building and utilizing libraries of reusable design components (e.g., Chaten) that can be easily integrated and customized by AI.

## Patterns & frameworks
-   **Swarm Mode:** This framework involves distributing a design task among multiple AI agents (e.g., six agents), allowing them to work in parallel on different parts of the design (e.g., two agents per screen for a three-screen app). The agents communicate to analyze the request, create a to-do list, pick a style guide, and distribute the work, significantly accelerating the design process.
-   **"Pen" (Platform-Agnostic Design Format):** Described as an "agentic PDF" for the AI era, Pen is a JSON-based file format that serves as a full, structured description of a design. It is built from the ground up for AI agents to read and write, ensuring platform agnosticism (e.g., convertible to Swift iOS, Kotlin, React Native, Figma, HTML). It acts as a central "source of truth" that can be stored in Git, enabling seamless collaboration between agents and humans across different tools and platforms.
-   **Visual Planning Mode:** This framework treats the Pencil canvas as a visual counterpart to "plan mode" in coding IDEs (like Cursor or Cloud Code). Instead of writing code directly, users visually plan, draw, and conceptualize designs. This approach encourages early and rapid ideation, allowing users to explore multiple design paths and refine their vision before committing to code generation, promoting an iterative and exploratory design process.
-   **Agentic Design Canvas Workflow:** This pattern emphasizes real-time, interactive collaboration between humans and AI agents on a visual canvas. Unlike typical byte-coding apps where users wait for the AI to finish, this workflow allows users to directly edit the design on the canvas *while* agents are actively working. This continuous interaction keeps the user "in the flow," making the design process more intuitive and efficient.
-   **Humanizing AI with Cursors:** A user experience design pattern where individual AI agents are visually represented by distinct cursors on the design canvas. This seemingly small detail makes the AI feel more present, relatable, and collaborative, enhancing the user's understanding of the AI's actions and making the interaction feel more "humanized."
