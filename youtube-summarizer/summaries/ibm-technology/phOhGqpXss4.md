# Predictive vs Generative AI: How They Work and When to Use Each

Video ID: `phOhGqpXss4`

## Summary
This video breaks down the fundamental differences between predictive AI and generative AI, explaining how each works, what kind of data they consume, and what they output. The presenter covers the core algorithms behind predictive AI (regression, classification, time series) and the architectures behind generative AI (transformers and diffusion models). Real-world use cases for both are explored, and the video concludes by showing how the two types of AI can complement each other rather than compete.

## Key insights
- **They answer different questions:** Predictive AI asks *"What will happen?"* and produces measurable outputs like numbers, categories, or probabilities. Generative AI asks *"What could this look like?"* and produces new content like text, images, or code.
- **Data types differ:** Predictive AI primarily consumes structured data (rows, columns, database tables), while generative AI consumes unstructured data (web text, images, code snippets).
- **Predictive AI is deterministic; generative AI is probabilistic:** The same input to a predictive model yields the same output every time, whereas generative models introduce randomness (controlled by a "temperature" setting), often producing different outputs for identical prompts.
- **LLMs are classified as generative:** Although large language models technically predict the next token at a mechanical level, their purpose and practical use place them firmly in the generative AI category.
- **Core predictive AI flavors:** Regression (predicting numbers), classification (predicting categories), and time series (predicting values that change over time), powered by algorithms like decision trees, random forests, gradient boosting, and LSTMs.
- **Generative AI architectures:** Text generation relies on the transformer architecture and its attention mechanism; image generation relies on diffusion models, which learn to progressively remove noise from images.
- **Common use cases — Predictive AI:** Fraud detection, demand forecasting, predictive maintenance, and credit scoring.
- **Common use cases — Generative AI:** Content creation, code assistance, conversational AI (chatbots), and document summarization.
- **The two types work best together:** A predictive model can identify which customers are likely to churn, then a generative model can draft personalized retention emails for each — prediction identifies the problem, generation crafts the response.
- **Generative AI can support predictive AI:** Generative models can create synthetic training data for predictive models when real data is scarce or sensitive.
