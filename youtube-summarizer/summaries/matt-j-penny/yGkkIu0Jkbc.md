# Stop Paying Anthropic $200/month for Claude Code (2 Alternatives)

Video ID: `yGkkIu0Jkbc`

## Summary
This video addresses the significant cost and usage limitations power users face with Claude Code, proposing two free alternatives that replicate 90% of its functionality. The core argument highlights that Claude Code's power comes from an expensive AI model ("brain") and a free software component ("agentic loop" or "harness"). By replacing Claude Code's proprietary agentic loop with a free, open-source alternative and connecting to pay-per-use AI models, users can drastically reduce costs, gain flexibility in model choice, and eliminate usage limits. This approach is highly relevant for developers, product managers, and business owners leveraging AI for coding who want to optimize expenses and maintain control over their tools without needing expensive local hardware.

## Key insights
- **High Cost of Claude Code:** Power users can quickly scale from $20/month to $100/month, and even $200/month for Claude Code, yet still frequently hit usage limits and receive interruption notifications.
- **Focus on Cloud-Based Models:** The video explicitly states it's *not* about running open-source models locally, as this requires a "seriously beefy computer" (e.g., a $2-3K GPU) that 99% of users don't possess.
- **Two Components of AI Assistants:**
    1.  **AI Models (the "brain"):** Such as Opus 4.6, Sonnet 4.6, Haiku, which are the expensive part, consuming credits and hitting limits.
    2.  **Agentic Loop / Harness (the "body"):** This is the software that enables the AI to perform actions like calling tools, using skills, and doing web fetches. This part is essentially free software.
- **Key Unlock:** The central idea is to replace Claude Code's proprietary agentic loop with a free alternative that allows users to "bring their own model" and pay only for AI model usage, avoiding fixed subscriptions and limits.
- **Alternative 1: Open Code:**
    - An open-source, free-to-install desktop application.
    - Offers features like file editing, agents, tools, skills, and MCPS, similar to Claude Code.
    - Can connect to free, promotional models (often lower quality or temporary) or, for better results, paid models via a marketplace like Open Router.
- **Alternative 2: Rode (Rue Code):**
    - A free VS Code extension, easy to install.
    - Provides comparable functionality including file editing, terminal commands, agents, and MCP agents.
    - Features specific modes, like "Architect mode" for planning and "Code mode" for coding.
    - It is battle-tested and used internally by large companies like Amazon for software engineering.
- **Open Router:** Recommended as a marketplace for AI models, providing access to hundreds of different models (Claude, OpenAI, Gemini, Mistral, Deep Seek, etc.) through a single account and API key.
- **Cost Savings & Flexibility:** Both Open Code and Rode are free to install. Users only pay for the actual AI model usage (tokens) through services like Open Router. This "pay-per-use" model is often "significantly cheaper" than fixed subscriptions and provides "incredible flexibility" to choose the right model (e.g., Deep Seek 3.2, which is "10 times cheaper" than Claude Opus) for a specific task, optimizing cost and quality.
- **No More Limits:** This approach eliminates hard usage limits, allowing users to consume as much as needed without interruption.
- **Free Resource:** The presenter offers a free list of over 600 Claude Code skills that are compatible with Open Code, Rode, and OpenClaw.

## Patterns & frameworks
- **Agentic Loop / Harness:** This framework describes the "body" or operational software layer of an AI assistant that facilitates actions (tool use, skill execution, web fetches), distinct from the "brain" (the AI model itself). The video's core strategy is to replace the proprietary agentic loop with free alternatives, unlocking cost savings.
- **"Brain and Body" Analogy:** A mental model used to explain the two core components of an AI coding assistant: the AI model as the expensive "brain" providing intelligence, and the agentic loop/harness as the free "body" executing tasks.
- **Model Marketplace (e.g., Open Router):** A repeatable pattern for accessing and managing multiple AI models from various providers through a single platform and API key. This enables users to select the most appropriate (and cost-effective) model for each specific task.
- **Architect/Code Modes (in Rue Code):** A structured approach to software development within an AI assistant, segmenting the workflow into distinct planning (Architect) and implementation (Code) phases.
