# Tridion — Changes 2026-06-16

### What's new - Docs

## What Changed
The Tridion website's main landing page shifted focus entirely from **Tridion Docs 15** (a CCMS/structured content product) to **Tridion Sites** (a web CMS product), representing a significant product spotlight change.

## Notable Updates
- **New product focus: Tridion Sites** — the page now promotes "One CMS for all your web needs" with global website management, replacing the previous Tridion Docs 15 feature showcase.
- **New product tiers introduced** — a "Compare Products" table now displays three editions: **Tridion Sites Essential**, **Tridion Sites Team**, and **Tridion Sites Enterprise**, with detailed feature comparisons (users, websites, SSO, personalization, GraphQL quotas, integrations, etc.).
- **Pricing model visible** — all three tiers show "Contact" for pricing, but key limits are now public (e.g., Essential/Team: 10 included users, max 25; Enterprise: 25 included, unlimited max; Essential/Team: 2.4M GraphQL queries included, max 6M; Enterprise: unlimited).
- **Enterprise-only features called out** — page regions, Instant Site, Personalization/A/B testing (on-demand), Semantic AI, unlimited integrations, and sustainable energy hosting are exclusive to Enterprise.
- **New "Three easy steps" onboarding pitch** — Adopt, Integrate, Deliver framing added, emphasizing hosted/managed offering (Essential Edition) and localization tech (Team Edition).
- **Customer logos added** — NH Hotel Group, KONE, Cambridge University Press, Jotun, Radisson Hotels, Starlux Airlines, Yale University now featured.
- **Contact form embedded** — a full lead-gen form (name, email, company, job title, country) is now inline on the page.
- **Removed:** All Tridion Docs 15/15.2 content (AI authoring features, Hexahops, Trustable Chat, Draft Companion, Organize Space, demo videos, feature comparison table) is gone from this page.
