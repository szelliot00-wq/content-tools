# oXygen XML Editor — Changes 2026-06-14

### Release history

## What Changed

The Oxygen XML Editor **feature matrix page has been massively simplified** — the entire detailed product comparison table (hundreds of feature rows spanning XML editing, DITA, publishing, validation, and more) along with the product selector checkboxes and XML Editor sub-navigation have been removed, cutting the page from ~2,975 lines to ~1,612.

## Notable Updates

- **Feature matrix table removed**: The full side-by-side comparison of XML Editor, XML Developer, JSON Editor, XML Author, and Web Author features (with Enterprise/Professional/Academic tier breakdowns) is gone.
- **XML Editor sub-navigation removed**: Page-level links for Overview, What's New, Get a Trial, Features, Downloads, Pricing, Documentation, and Videos no longer appear on this page.
- **Feature category sidebar removed**: The deep hierarchical sidebar (Visual XML Editors, XSL/XSLT, XML Schema, WSDL, JSON, YAML, OpenAPI/AsyncAPI, Relax NG, XQuery, XProc, SVN Client, etc.) has been stripped out.
- **Product selector removed**: The interactive checkboxes for toggling which products to compare in the matrix are gone.
- The remaining new content only shows the top-level site navigation — whatever replaced the matrix content was not captured in this diff, so the page may have been restructured into a different format (e.g., separate per-product pages or a new comparison UI).
