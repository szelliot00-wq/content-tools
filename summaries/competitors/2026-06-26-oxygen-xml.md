# oXygen XML Editor — Changes 2026-06-26

### Pricing

## What Changed
Oxygen XML's business license shop page gained a new card/list view toggle for the product listing, and the product entries were restructured with more explicit visual grouping — but prices and the product lineup itself are unchanged.

## Notable Updates
- **New product listing view toggle**: A "Card view" / "List view" switcher was added to the Business License shop page, letting users choose how products are displayed.
- **Restructured product layout**: The "New Licenses" section now wraps each product (Oxygen XML Editor, Developer, Author, etc.) in its own grouped container with licensing type sub-sections (User-based, Floating), making the hierarchy more explicit — previously the items were flatter in the DOM.
- **Navigation tabs also restructured**: The "New Licenses / Upgrades / SMP Renewals" tab links are now nested inside a grouped container rather than sitting as direct siblings, suggesting a UI refresh to that tab bar.
- **No pricing changes**: All prices (e.g. XML Editor Enterprise User-based €1,465 / €1,678; Floating €4,397 / €5,037; Feedback tiers €350–€1,890/month) remain identical to before.
- **No new products**: The full product lineup (XML Editor, Developer, Author, JSON Editor, Web Author, Content Fusion, Feedback, AI Positron, Publishing Engine, WebHelp, PDF Chemistry, Scripting) is unchanged.
