### What's new - Docs

## What Changed
The page shifted focus from the **Tridion Sites** CMS product to **Tridion Docs 15/15.2**, a Component Content Management System (CCMS). The entire main content area was replaced with a product update announcement for Tridion Docs 15 and its latest release, v15.2.

## Notable Updates
- **New product spotlight: Tridion Docs 15.2** — The page now promotes "the best CCMS on the market," highlighting major advancements in AI-assisted authoring, headless delivery, and project management.
- **Tridion Docs Genius** — New AI-driven knowledge portal introduced, featuring "Trustable Chat" (RAG-based chat grounded in documentation) and "Hexahops" (a honeycomb-style visual navigation paradigm).
- **Draft Companion (GenAI)** — New generative AI tool in Draft Space that helps authors simplify, rephrase, fix grammar, and summarize content while keeping authors in control.
- **Dynamic Experience Delivery (DXD)** — Enhanced headless content delivery with GraphQL support for DITA topics and low-code multichannel publishing.
- **Organize Space** — Replaces the old Content Manager Web Client with a new, browser-optimized UI for publication management, accessible on desktop, tablet, and mobile.
- **Publication Hub** — New project management area giving users an overview of all ongoing projects, task assignments, and work status; integrates with Jira and Workfront via API.
- **Metrics & ROI reporting** — New out-of-the-box content reuse reports exportable to BI tools to measure efficiency and justify CCMS investment.
- **Modernized security** — OpenID Connect authentication added, with support for 2FA, granular permissions, and full username display across all spaces.
- **Extended platform support** — Now supports oXygen 27, XMetaL 19, Windows Server 2022, .NET 8, Java 21, and Solr 9.4.1.
- **OpenAPI adoption** — New Tridion Docs OpenAPI standard enables language-agnostic, platform-independent integrations with HATEOAS links and JSON models.
- **Removed: Tridion Sites product comparison table** — The tiered pricing/feature comparison (Essential, Team, Enterprise) and associated contact form were removed from this page entirely.
