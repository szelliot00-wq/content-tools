### Release history

## What Changed
The competitor has released multiple significant updates across their product line, introducing major advancements in AI capabilities, a new "Personal Workspace" feature set, and substantial enhancements to DITA content management and web authoring tools.

## Notable Updates
-   **Enhanced AI Capabilities:** Oxygen AI Positron Assistant (versions 5.0 and 7.0) now supports OpenAI's latest GPT-5 models, offers AI-powered automatic fixes for XML validation, allows tool invocation via Model Context Protocol Servers, and includes new chat modes, a `search_in_project` tool, and improved attachment handling (visual chips, screenshots, .docx). It also enables generating initial content for various XML schema types and DTDs from text descriptions and provides DITA-specific content transformation actions.
-   **Oxygen XML Web Author 28.0.0:** This update focuses on AI-assisted authoring, DITA-centric workflows, enhanced enterprise integration with deeper AI Positron 8.0 integration, and strengthened security for large organizations.
-   **New "Personal Workspace" 8.2:** Introduced as a new feature set, it provides enhanced publication structure editing, detailed commit history, easier branch management, selective file commits, and the ability to create targeted review tasks.
-   **Oxygen Content Fusion 8.1:** This version streamlines DITA content management by allowing the inclusion of publications from existing DITA-OT project files, offering flexible project creation without immediate Git integration, and supporting predefined templates for faster setup.
