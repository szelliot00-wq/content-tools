# oXygen XML Editor — Changes 2026-06-13

### Product overview

## What Changed
The Oxygen XML Editor's feature matrix page was significantly stripped down — the full product comparison table and the detailed sidebar navigation were removed, reducing the page from ~2,975 lines to ~420 lines.

## Notable Updates
- **Feature matrix table removed**: The comprehensive side-by-side comparison of all five products (XML Editor, XML Developer, JSON Editor, XML Author, Web Author) — with hundreds of rows covering XML editing, DITA, collaboration, AI tools, publishing, schema tools, etc. — no longer appears on the page.
- **Sidebar navigation simplified**: The extensive sub-menu under Oxygen XML Editor (WebHelp, XSL/XSLT, XML Schema, WSDL, Ant, Relax NG, XQuery, JSON, JSON Schema, YAML, OpenAPI/AsyncAPI, HTML, Markdown, XProc, XML Databases, CMS Integration, Compare Tools, SVN Client, and more) has been removed; only a stub of the "Visual XML Editors" section remains visible.
- The top-level navigation tabs (Products, Shop, Resources, Support, Company) and the XML Editor sub-nav (Overview, What's New, Features, Pricing, etc.) are unchanged.
