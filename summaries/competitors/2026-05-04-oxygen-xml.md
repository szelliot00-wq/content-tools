# oXygen XML Editor — Changes 2026-05-04

### Product overview

## What Changed
Oxygen XML Editor's website has been significantly restructured with a modernized layout and updated navigation, while also introducing new pricing information and enhanced product positioning on the homepage.

## Notable Updates
- **Pricing now visible on homepage:** A "Buy (from €34/month)" call-to-action button is now prominently displayed on the main banner, making subscription pricing immediately visible to visitors.
- **New hero banner:** The homepage now features a structured banner with quick links to key capabilities (Single-Source Publishing, JSON Tools, Structured XML Editing, Database and CMS, XML Tools) and a clearer "Download / Buy / Videos" CTA layout.
- **AI Positron highlighted prominently:** A dedicated banner for "AI Positron" has been added to the homepage, elevating the AI assistant feature as a core product offering.
- **New product equation banner:** A new section visually communicates that "XML Editor = XML Author + XML Developer," clarifying the product bundling to prospective buyers.
- **Login functionality added:** A "Login" link has been added to the main navigation, suggesting a user account/portal feature is now available.
- **New feature pages added:** "Authoring," "Review" (with sub-pages for reviewing from Oxygen XML Editor/Author and DITA/Git projects), "Publishing," and "Content Contribution" have been added as distinct feature sections in the navigation.
- **Generic Visual XML Editor expanded:** Now includes sub-pages for "Review Tools," "Profiling/Conditional Text," and "Constant Validation."

## Notable Updates
- **New AI Feature**: "Oxygen AI Positron Assistant Enterprise" has been added as a dedicated feature category under AI Tools, available across XML Editor, XML Developer, and XML Author products

- **New Review/Collaboration Section**: Expanded review features now include "Review from Oxygen XML Editor/Author" and "Review from a DITA/Git Project" as distinct sub-categories
- **New Content Contribution Category**: A "Content Contribution" section has been added under publishing/review features
- **Updated Product Tiers**: The matrix now shows "Enterprise" and "Professional" tier distinctions more explicitly across products, including a "Professional" tier for Web Author
- **Navigation Expanded**: New menu items added including "Publishing," "Content Contribution," and separate "Review" sections distinct from the previous combined authoring/review structure
- **Web Author Feature Annotations**: Many existing features now explicitly note web browser availability (or lack thereof) with Web Author-specific indicators throughout the matrix

- **Oxygen Scripting pricing** for new licenses now omits explicit per-core price display in the new layout (prices for 2/4/8 core options are still present but shown differently)
- **Oxygen Feedback** Cloud and Enterprise sections now display product icons/images ("Feedback Cloud" and "Feedback Enterprise" images) alongside their labels
- All **prices remain unchanged** across all products (XML Editor, Developer, Author, JSON Editor, Publishing Engine, WebHelp, PDF Chemistry, Scripting, Feedback)

### Release history

## What Changed
The Release History page has been significantly restructured from a long-form text page into a structured, interactive timeline format where each version entry now includes direct "Click to see what is new" links to dedicated release notes pages for individual products.

## Notable Updates
- Each release entry now has clickable links to product-specific "what's new" pages (e.g., separate links for Editor, Author, Developer, Chemistry, WebHelp, Publishing Engine, Scripting, JSON Editor, SDK, Web Author, AI Positron, Content Fusion, Feedback Cloud, Feedback Enterprise)
- The detailed descriptive text for each release has been removed from the main timeline; content is now accessed via the linked detail pages
- A **Login** button/link has been added to the site navigation
- Older versions (pre-version 8.0, going back to 1.0) now also have "what's new" links, whereas previously many of those entries had no descriptions at all
- The page now uses a structured, consistent time-based layout with `<time>` elements for each release entry
- Company name in the footer updated from "SyncRO Soft SRL" to "Syncro Soft SRL" (capitalization change)
