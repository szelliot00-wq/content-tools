# Syncro Soft Announces the Release of Oxygen AI Positron 8.3

Source: 

## Summary
Syncro Soft has released Oxygen AI Positron 8.3, a major update to its AI plugin suite for the Oxygen XML product family. The release focuses on enterprise AI governance, a new project-aware development agent, expanded model support (including GPT-5.5 and Claude Opus 4.8), and new XML automation capabilities. It targets technical writers, XML developers, and enterprise administrators who need more control and transparency over AI-assisted workflows.

## Key takeaways
- **AI Governance via Event Hooks:** Organizations can now intercept, audit, and block AI-driven actions using a new hook system, with an optional add-on to trigger external scripts for compliance and analytics pipelines.
- **Dev Agent Chat Mode:** A new project-aware AI assistant handles XSLT, Schematron, XPath, and other XML development tasks, proposing and executing project-wide changes while keeping the user in review control.
- **Specialized XML Skills:** Version 8.3 adds targeted skills for XSLT, XSD, Schematron, and XSpec to improve code suggestion accuracy, and custom skills can now be distributed as `.skill` ZIP packages.
- **Expanded Model Support:** New models include GPT-5.4/5.5 Thinking, Claude Opus 4.8 (with Thinking variant), and Gemini 3.5 Flash, with reasoning output now visible in a collapsible chat section.
- **Broader Enterprise Connectivity:** New connectors support OpenAI Responses API, MCP HTTP servers with OAuth, and AWS Bedrock with Profile/SSO authentication.
- **Automatic Conversation Compaction:** Long AI sessions are kept alive by auto-summarizing older context when usage hits 70%, preventing context-limit interruptions.
- **Batch Validation Fixes:** Multiple validation errors in the Results pane can now be resolved in a single AI-assisted action, boosting productivity on large XML documents.
- **New XPath AI Functions:** Two new extension functions (`ai:chat()` and `ai:resolve-action()`) and an `evaluate_xpath` tool enable deeper AI integration into XPath-based automation workflows.
