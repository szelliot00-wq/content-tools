# Oxygen Content Fusion 9.0 is Now Available!

Source: news_content_fusion_9_0

## Summary
Syncro Soft has announced the immediate availability of Oxygen Content Fusion 9.0, its web-based collaboration platform for content creation. This release significantly enhances end-to-end DITA collaboration, particularly for technical communication teams managing complex, versioned documentation, by introducing multi-branch publishing pipelines, more flexible review workflows, and advanced AI-assisted authoring. The update aims to streamline review processes and accelerate content creation and maintenance through features like improved project workflows, an enhanced visual editor, and a more powerful Oxygen AI Positron integration.

## Key takeaways
- Oxygen Content Fusion 9.0 significantly strengthens DITA collaboration for technical communication teams, especially those managing complex, versioned documentation projects.
- Key new features include multi-branch publishing pipelines, which allow organizations to manage parallel product versions or regional variants, and more flexible review workflows, enabling iterative review cycles.
- The platform now integrates an enhanced Oxygen AI Positron for Content Fusion, offering more powerful, project-aware AI assistance with an expanded set of models and is available at no extra cost with the Content Fusion license.
- The visual editing experience has been improved with features such as easier table resizing, direct commenting from a floating toolbar, better Markdown handling, and native media controls for DITA 2.0 video/audio elements.
- Project workflows are enhanced with better control over context for reviewers, and the ability to easily create review tasks from publication structures, making content review and publication more efficient.
