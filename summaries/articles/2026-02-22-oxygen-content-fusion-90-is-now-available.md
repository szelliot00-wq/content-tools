# Oxygen Content Fusion 9.0 is Now Available!

Source: 

## Summary
Syncro Soft has released Oxygen Content Fusion 9.0, a new version of its web-based collaboration platform designed for technical communication teams managing complex, versioned documentation. This release significantly strengthens end-to-end DITA collaboration by introducing multi-branch publishing pipelines, more flexible review workflows, and enhanced AI-assisted authoring through Oxygen AI Positron. Version 9.0 aims to streamline review processes and accelerate content creation and maintenance by leveraging AI and improving project management capabilities.

## Key takeaways
- Oxygen Content Fusion 9.0 is a web-based collaboration platform by Syncro Soft, specifically tailored for technical communication teams managing complex, versioned DITA documentation.
- The release introduces multi-branch publishing pipelines, enabling organizations to manage parallel product versions, regional variants, or long-lived maintenance branches more effectively.
- It provides more flexible review workflows, allowing for iterative review cycles and the reopening of previously integrated tasks, thus streamlining late-stage changes and ad-hoc reviews.
- The platform features an enhanced visual editing experience, incorporating updates from Oxygen XML Web Author 28.0, with improvements in table handling, in-context commenting, Markdown content rendering, and multimedia controls.
- Oxygen AI Positron for Content Fusion is deeply integrated, offering an expanded set of AI models and project-aware assistance to accelerate content creation and maintenance, available at no extra cost with the Content Fusion license.
