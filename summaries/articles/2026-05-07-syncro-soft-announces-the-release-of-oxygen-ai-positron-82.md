# Syncro Soft Announces the Release of Oxygen AI Positron 8.2

Source: 

## Summary
Syncro Soft has released Oxygen AI Positron 8.2, a major update to its AI-assisted authoring plugins for the Oxygen XML suite of tools. The release focuses on three core areas: extensibility through reusable skills, improved context awareness during AI interactions, and greater transparency in how the AI reasons and responds. The update is available across all major Oxygen XML products, including the desktop, Eclipse, web, and collaboration platform versions.

## Key Takeaways
- **Reusable Skills:** A new skills system based on Anthropic Agent Skills allows organizations to build specialized AI modules for repetitive workflows, external API integrations, and domain-specific processes, with several built-in skills included out of the box.
- **Advanced Reasoning Models:** Support for Claude Sonnet 4.6 Thinking and Claude Opus 4.6 Thinking models has been added, with a collapsible real-time summary of the model's reasoning process displayed in the chat interface for greater transparency.
- **Smarter Context Awareness:** The AI assistant better tracks the user's current working context across ongoing conversations, including when files or content selections change mid-chat.
- **Context Window Indicator:** A new visual indicator shows users how much of the model's available context window is being used, helping manage complex interactions more effectively.
- **Expanded File Analysis:** Users can now analyze PDF, Word, Excel, PowerPoint, image, and SVG files directly within AI interactions without manual attachment steps.
- **Enterprise Enhancements:** New support for configuring reasoning effort levels on Microsoft Azure and for Claude deployments hosted in Azure AI Foundry strengthens enterprise deployment options.
