# OpenTTD 16.0-Beta1

Source: https://www.openttd.org/news/2026/06/25/openttd-16-0-beta1

## Summary
OpenTTD 16.0-beta1 has been released for public testing, bringing a host of new features to the open-source transport simulation game. The release includes notable improvements to map generation and other gameplay elements. Alongside the beta, the community has launched a competition to design the title game (main menu background) for the final 16.0 release.

## Key takeaways
- OpenTTD 16.0-beta1 is now available for download and testing.
- Map generation has been significantly improved: desert rocks, better coastal edges, improved lighthouse spawning on large maps, transmitters restricted to post-1920, and custom average land height control.
- A Title Game Competition is running on TT-Forums, inviting the community to design and vote on the main menu game for the 16.0 release.
- Bug reports for the beta can be submitted via the official bug tracker.
- The stable release remains 15.3, with 16.0-beta1 available as a testing build.
