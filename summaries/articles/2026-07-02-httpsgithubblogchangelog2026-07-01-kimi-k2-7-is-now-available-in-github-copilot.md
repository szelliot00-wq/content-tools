# Kimi K2.7 Code is generally available in GitHub Copilot

Source: https://github.blog/changelog/2026-07-01-kimi-k2-7-is-now-available-in-github-copilot/

## Summary
GitHub Copilot has added Kimi K2.7 Code as a generally available model, marking it as the first open-weight model selectable in Copilot's model picker. It is hosted by GitHub on Microsoft Azure and billed at provider list pricing. The rollout is initially targeting Copilot Pro, Pro+, and Max plans across a range of IDEs and surfaces, with Business and Enterprise support coming in the following weeks.

## Key takeaways
- Kimi K2.7 Code is the first open-weight model available in the GitHub Copilot model picker, offering users a lower-cost coding option.
- It is hosted on Microsoft Azure and billed at provider list pricing under usage-based billing.
- Available in VS Code (1.127.0+), Visual Studio (17.14.6+), JetBrains (1.9.1-251+), Copilot CLI, GitHub Mobile, Xcode, Eclipse, and github.com.
- For Copilot Business and Enterprise, the model is off by default and must be explicitly enabled by plan administrators.
- Administrators are advised to review the open-weight model against their organization's security, compliance, and data-governance policies before enabling it.
