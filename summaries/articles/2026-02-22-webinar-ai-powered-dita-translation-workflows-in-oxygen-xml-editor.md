# Webinar: AI-powered DITA Translation Workflows in Oxygen XML Editor

Source: 

## Summary
This article announces a free live webinar on February 4, focusing on AI-powered DITA translation workflows within the Oxygen XML Editor. Presented by Sorin Carbunaru, the session will demonstrate how the Fluenta DITA Translation and Oxygen AI Translator add-ons work together to streamline DITA project translation by simplifying XLIFF generation and integrating AI-driven translation capabilities. Attendees will learn practical steps for organizing DITA projects, managing translation packages, and generating efficient translations.

## Key takeaways
- The webinar introduces an AI-powered DITA translation workflow using Oxygen XML Editor.
- It highlights the combined use of two add-ons: Fluenta DITA Translation and Oxygen AI Translator.
- Fluenta DITA Translation simplifies the process by generating a single, unified XLIFF file for Language Service Providers (LSPs).
- Oxygen AI Translator adds AI translation capabilities to XLIFF-based projects, aiming for speed and quality.
- The session will include practical demonstrations on structuring DITA projects, managing translation packages, and leveraging AI for efficient translation.
- The event is free but requires registration.
