# Dear Microsoft, enough is enough

Source: https://www.politico.eu/sponsored-content/dear-microsoft-enough-is-enough/

## Summary
This is a sponsored open letter published on POLITICO by the Browser Choice Alliance, addressed to Microsoft CEO Satya Nadella. The coalition of browser developers accuses Microsoft of using its dominant Windows OS position to unfairly push users toward its Edge browser through manipulative and restrictive tactics. They call on Microsoft to immediately adopt fair competition practices that respect user choice globally.

Note: This is paid/sponsored content, not independent journalism.

## Key takeaways
- Microsoft is accused of using coercive OEM rebates, dark patterns, and manipulative UI design to steer users away from competing browsers and toward Edge.
- Specific tactics cited include: preventing Edge uninstallation, ignoring users' default browser settings in Teams/Outlook, using OS updates to revert browser defaults, and blocking third-party browsers in Windows S mode.
- The Alliance demands Microsoft allow one-click default browser switching, open all web links in the user's chosen browser, end manipulative Edge-promoting banners, and let OEMs preinstall rival browsers freely.
- The letter frames browser choice as increasingly critical in the AI era, where PCs are central to high-value tasks like coding and research.
- The piece is sponsored content/political advertising — the Browser Choice Alliance represents competing browser vendors with a direct commercial interest in the outcome.
