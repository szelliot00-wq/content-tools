# The Sneakerweb

Source: https://sneakerweb.org/

## Summary
The Sneakerweb is a peer-to-peer web publishing protocol that operates independently of traditional internet infrastructure like DNS servers, domain registrars, and web hosts. Websites are stored directly on user devices and shared via physical storage media, creating a parallel web that requires no central permissions or gatekeepers. The project provides a CLI tool and a formal protocol specification, with ties to the Willow protocol.

## Key takeaways
- The Sneakerweb enables web publishing without any centralized infrastructure — no DNS, no domain registrars, no web hosts.
- Websites are stored on users' own devices and transferred through physical media (e.g., USB drives, SD cards).
- It is a permissionless, peer-to-peer system, meaning anyone can publish without approval from a central authority.
- A CLI tool is available for users to participate in the network.
- The protocol is fully specified and appears to build on or relate to the Willow protocol.
