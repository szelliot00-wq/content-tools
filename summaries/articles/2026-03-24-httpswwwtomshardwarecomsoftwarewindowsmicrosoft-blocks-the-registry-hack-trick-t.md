# Microsoft blocks trick to unlock native NVMe driver, but workarounds still exist

Source: https://www.tomshardware.com/software/windows/microsoft-blocks-the-registry-hack-trick-that-unlocked-native-nvme-performance-on-windows-11

## Summary
Microsoft has implemented a block against a specific "trick" previously used to unlock the native NVMe driver and its associated performance benefits on Windows 11. This action prevents users from utilizing a known registry hack or similar method to improve NVMe storage performance. However, despite Microsoft's intervention, alternative workarounds and methods are reportedly still available for users aiming to enable native NVMe driver functionality.

## Key takeaways
- Microsoft has taken steps to block a specific "trick" or method that previously allowed users to enable the native NVMe driver in Windows 11.
- This blocked method was primarily used to unlock enhanced performance for NVMe SSDs.
- While the original trick is no longer functional, alternative workarounds are still available for users who wish to achieve native NVMe performance.
