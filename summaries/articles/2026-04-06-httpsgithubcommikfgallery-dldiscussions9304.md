# Media scraper Gallery-dl is moving to Codeberg after receiving a DMCA notice

Source: https://github.com/mikf/gallery-dl/discussions/9304

## Summary
The popular media scraper project, `gallery-dl`, is relocating its development from GitHub to Codeberg after receiving a DMCA takedown notice. The notice targeted components related to YouTube media extraction, which GitHub complied with by disabling access to the identified files. Developer Mikf decided to move, stating that GitHub's compliance makes it an unsuitable platform for a project frequently dealing with content potentially subject to legal challenges, while ensuring the project's continued availability on an open-source platform.

## Key takeaways
-   `gallery-dl`, a media scraper and downloader, is moving its entire development and repository from GitHub to Codeberg.
-   The migration was triggered by a DMCA takedown notice that targeted the `youtube-dl` fork used within `gallery-dl` for YouTube media extraction.
-   GitHub complied with the DMCA notice by disabling access to the specific files, leading the developer, Mikf, to believe GitHub is an unreliable host for such projects.
-   Codeberg, an open-source platform running Gitea, was chosen for its alignment with free software principles and perceived better suitability for projects dealing with content potentially subject to takedowns.
-   The existing GitHub repository will be maintained as a read-only archive, directing users to the new Codeberg location for all future development and contributions.
