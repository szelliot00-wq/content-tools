# Syncro Soft announces the release of version 28.1 for the Oxygen XML suite of products!

Source: 

## Summary
Syncro Soft has released version 28.1 of its Oxygen XML suite of products, including the flagship XML Editor, Web Author, and XML Author. The update introduces significant AI-powered enhancements, most notably the "Planner" chat mode in Oxygen AI Positron, improved sensitive data masking, and the new Oxygen AI Translator add-on for XLIFF workflows. This release also features built-in support for S1000D issue 4.x, various DITA editing and PDF publishing improvements, and general enhancements across development, collaboration, and scripting tools.

## Key takeaways
- **Major AI-powered workflow enhancements:** The Oxygen AI Positron add-on introduces a "Planner" chat mode for multi-step task automation and improved sensitive data masking. The new Oxygen AI Translator add-on provides enterprise-grade AI translation for XLIFF workflows, featuring quality control, translation memory integration, and smart caching.
- **Built-in S1000D 4.x support:** The suite now offers comprehensive, built-in support for S1000D issue 4.x content, including schema-guided editing, visual authoring, BREX module editing, Smart Paste, and ready-to-use templates, benefiting aerospace and defense users.
- **Improved DITA publishing and authoring:** Enhancements include form controls for editing change history in DITA prolog sections, character-level font fallback for PDF publishing, and new features in PDF Chemistry and XML WebHelp for more flexible and accurate output generation.
- **Developer and collaboration tool advancements:** XProc 3.0 receives enhancements, file comparison tools are upgraded with customizable reports, and the Git Client add-on includes a new "Explain commit" AI action. XML Scripting now supports multiple configuration formats and comprehensive metadata tracking for comparison reports.
- **Suite-wide consistency and core updates:** Many new features and enhancements are integrated across the entire Oxygen XML product suite, including Web Author, Author, Developer, and JSON Editor. The release also brings important component updates (OpenJDK, DITA-OT, XML Calabash, Eclipse), bug fixes, and security improvements.
