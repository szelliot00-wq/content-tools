# Webinar: Streamlining S1000D Authoring with Oxygen XML Editor

Source: 

## Summary
This article is an announcement for a free, one-hour webinar scheduled for May 20, focused on simplifying S1000D authoring using Oxygen XML Editor. The session will be led by Octavian Nadolu, Product Manager at Syncro Soft, who will demonstrate key features and workflows across multiple Oxygen platforms. The webinar will also highlight how AI-powered tools within Oxygen can enhance productivity and reduce repetitive authoring tasks.

## Key takeaways
- The webinar takes place on May 20 and is free to attend, though registration is required.
- Topics covered include editing, validation, visual authoring, templates, BREX handling, and Smart Paste within Oxygen XML Editor.
- The session will demonstrate managing Data Modules and Publication Modules across Oxygen XML Editor Desktop, Web Author, Content Fusion, and the Eclipse plugin.
- AI-powered capabilities in Oxygen will be showcased as tools to reduce repetitive tasks and boost author productivity.
- The goal of the webinar is to show how Oxygen can reduce authoring friction and create a more consistent S1000D workflow.
