## Summary
TerminalPhone is a self-contained Bash script enabling anonymous, end-to-end encrypted voice and text communication between two parties. It functions like a walkie-talkie, where voice messages are recorded, compressed, encrypted, and sent as single units over the Tor network. This tool requires no server infrastructure, accounts, or phone numbers, using a Tor hidden service .onion address as the sole form of identity.

## Key takeaways
- TerminalPhone is a single, self-contained Bash script for secure communication.
- It provides anonymous, end-to-end encrypted voice and text communication.
- Communication operates in a walkie-talkie style, sending discrete encrypted voice messages.
- It leverages the Tor network for anonymity and transmission.
- There is no central server infrastructure, user accounts, or phone numbers required.
- A user's Tor hidden service .onion address serves as their identity.
