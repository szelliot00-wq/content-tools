# Tribblix: The retro Illumos distribution

Source: http://tribblix.org/

## Summary
Tribblix is a retro-styled Linux distribution built on the Illumos operating system kernel. The project is actively maintained, with recent milestone releases for both x86 and SPARC architectures. The site serves as a hub for downloading, installing, and learning to use the distribution.

## Key takeaways
- Milestone 40 was released on June 12th, 2026 for x86 hardware.
- A SPARC architecture ISO (m34) became available on April 21st, 2026.
- 32-bit hardware support has been fully removed from Tribblix.
- The project is open source and hosted on GitHub, with community contact via email.
