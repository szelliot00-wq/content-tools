# Webinar: AI-powered DITA Translation Workflows in Oxygen XML Editor

Source: news_webinars_february_4_2026

## Summary
This article announces a free live webinar titled "AI-powered DITA Translation Workflows in Oxygen XML Editor," scheduled for February 4th. Presented by Sorin Carbunaru, the webinar will demonstrate how two Oxygen XML Editor add-ons, Fluenta DITA Translation and Oxygen AI Translator, can be used together to enhance DITA project translation speed and efficiency. Fluenta simplifies the process by generating unified XLIFF files for Language Service Providers, while Oxygen AI Translator adds AI-driven translation capabilities to these XLIFF projects. The session will cover practical workflows, including DITA project organization and leveraging both add-ons for translation management and generation.

## Key takeaways
- The webinar will focus on integrating AI into DITA translation workflows using Oxygen XML Editor.
- It will showcase two specific Oxygen XML add-ons: Fluenta DITA Translation and Oxygen AI Translator.
- Fluenta DITA Translation helps streamline DITA projects by creating a single XLIFF file for translation.
- Oxygen AI Translator complements this by providing AI-powered translation for XLIFF-based projects, aiming for speed and quality.
- Attendees will learn practical workflows for structuring DITA projects, managing translation packages with Fluenta, and generating translations efficiently with Oxygen AI Translator.
- The event is free but requires registration.
