# I'm going to build my own OpenClaw, with blackjack and bun

Source: https://github.com/rcarmo/piclaw

## Summary
The article introduces "piclaw," an open-source project focused on building a customizable, small USB mechanical keyboard or macropad using a Raspberry Pi Pico. Inspired by "Claw" style keyboards but prioritizing simplicity, portability, and low power, it runs CircuitPython for easy key layout configuration. The project aims to provide a DIY-friendly solution, involving custom PCB design for mechanical switches, suitable for a few extra keys, a foot pedal, or a compact macropad. The title playfully uses a meme to express the desire for a personalized, feature-rich, yet straightforward build.

## Key takeaways
- The "piclaw" project aims to create a simple, open-source, and custom USB mechanical keyboard or macropad.
- It is based on the Raspberry Pi Pico microcontroller and runs CircuitPython for flexible key mapping.
- The design emphasizes simplicity, portability, low power consumption, and DIY-friendliness, including a custom PCB for mechanical switches.
- Inspired by more complex "Claw" keyboards, piclaw targets a smaller, more accessible form factor suitable for auxiliary keys or foot pedal applications.
- The project is currently a work-in-progress, with ongoing development for both hardware and firmware.
