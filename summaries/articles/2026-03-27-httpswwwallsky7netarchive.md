# The European AllSky7 fireball network

Source: https://www.allsky7.net/#archive

## Summary
The European AllSky7 network is a collaborative citizen science project that employs a dense array of all-sky cameras across Europe to observe meteors and fireballs. It automatically detects these events, calculates their atmospheric trajectories, and determines the orbits of meteoroids using common software and precise GPS time synchronization. The network aims to facilitate scientific research, aid in the search for meteorite falls, and contribute to public education regarding celestial phenomena.

## Key takeaways
-   **Purpose:** AllSky7 is a network of all-sky cameras dedicated to the automatic detection and study of meteors and fireballs.
-   **Geographic Scope:** It aims for dense coverage across Europe, with stations operated in numerous countries by volunteers and institutions.
-   **Technology:** The network utilizes common software for event detection, trajectory calculation, and orbit determination, enhanced by accurate GPS time synchronization for precise data.
-   **Collaborative Model:** It functions as a citizen science project, with stations maintained by volunteers, observatories, and research institutions.
-   **Scientific Objectives:** Primary goals include detecting fireballs, calculating trajectories and orbits, searching for meteorite falls, supporting scientific research, and public education.
